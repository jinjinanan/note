## 结构体

### 直接定义结构体

```Objective-C
struct MyDate1 {
    int year;
    int month;
    int day;
};
struct MyDate1 d1 = {2016, 1, 6};
```

### typedef 定义结构体与调用

```Objective-C
typedef struct {
    char *name;
    int tag;
} structExt;
structExt a = {@"1",1};
```

## @interface

1. 声明类:Class
2. 声明类别:Category
3. 声明扩展:Extension

### 声明:Class

```Objective-C
@interface SomeClass : NSObject <SomeDelegate>{
}
@end
```

### 声明:Category

```Objective-C
@interface ClassName(类别名){
}
@end
```

### 声明:Extension

```Objective-C
@interface ClassName(){
}
@end
```
