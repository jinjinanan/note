## 常用插件
	- Package Control 				包管理工具
	- OmniMarkupPreviewer 			markdown扩展插件
	- Side Bar						侧边栏目录插件
	- Themr							管理风格
	- CTags							点击函数名跳转定义插件
	- SublimeGit					GIT 工具


---

### Package Control
	shift + command + p 			模糊匹配
	command + p 					根据文件名打开文件

#### 安装插件
1. `shift + command + p`
2. 输入 `install`
3. 选择 `Install Package` 并 回车选中
4. 输入 插件名字
